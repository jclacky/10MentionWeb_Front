D- Alignement

Les élément sont organisés soit horizontalement par défaut soit verticalement.

  • si vos élément sont organisés horizontalement: l'axe principal (main axis) est l'axe horizontale, l'axe secondaire (cross axis) est l'axe verticale
  • si vos élément sont organisés verticalement : l'axe principal (main axis) est l'axe vertical, l'axe secondaire (cross axis )est l'axe horizontale

Alignement sur l'axe pricipal

Afin d'aligner nos éléments sur l'axe principal on utilise la propriété du flex dans un Justify-content

Cette propriété gère la distribution des emplacement des flex-items suivant la place disponible dans le flex-conteiner. Elle le fait, selon le positionnement de l'axe principal

  • Flex-start: aligner au début (par défaurt)
  • flex-end : Aligner à la fin
  • Center : Aligner au centre
  • space-between : les éléments sont étirés sur tout l'axe (il y a de l'espace entre eux )
  • space-around : les élément sont étirés sur sur tout l'axe, mais ils laissent aussi de l'espace sur les extrémités
  • space-evenly : les éléments sont répartient uniformément dans le conteneur d'alignement le long de l'axe principal

E - Retour à la ligne

Par défaut, les blocs essaient de rester sur la m^me ligne même s'ils n'ont pas la places (ce qui provoque parfois des problèmes de conception ). Si vous le souhaitez, vous pouvez demander au blocx de passer à la lignes lorsqu'ils n'ont plus de place en utilisant la propriété Flex-wrap

Elle peut prendre les valeurs

  • Nowrap: Pas de retour à la ligne (la valeur par défaut)
  • wrap : les élélements vont à la ligne lorsqu'ils n'ont plus de place
  • wrap-revese: Les éléments vont à la ligne losqu'ils n'ont plus de place en sens inverse

F - Répartir plusieurs lignes

Si vous avez plusieurs ligne dans votre flexbox,vous pouvez choisir comment celles-ci seront réparties avec align-content. Cette propriété permet de gérer l'alignement des différents lignes (ou colonnes) les par rapports au autres. Cette propriété n'aura d'éffet que si le conteneur flexible posséde plusieurs lignes(si l'axe principale est horizontal) ou plusieurs colonnes ( si l'axe principle est vertical) et que si le flex-wrap est présent.

Les valeurs de cette propriété :

  • flex-start : les éléments sont placés au début
  • flex-end : les élélements sont placés à la fin
  • center : les éléments sont placés au centre
  • space-between : les éléments sont séparés avec de l'éspace entre eux
  • space-around : les éléments sont séparés avec de l'éspace entre eux, mais il y'a aussi de l'éspace au début et à la fin
  • space-evenly : Les éléments sont répartis uniformément dans le conteneur d'alignement le long de l'axe principal.
  • stretch (par défaut) : les éléments s'étirent pour occuper tout l'éspace
