C - Direction

Flexbox nous permet d'agencer éléments dans le sens que l'on veut. Avec flex-direction on peut les positionner verticalement ou encore les inverser. il peut prendre les valeurs suivvantes :
==> row : organisés sur une ligne (par défaut)
==> row-reverse : organisés sur une ligne en ordre inversé
==> Colomn ==> colonm-reverse: organisés sur une colonne, lais en ordre inversé
